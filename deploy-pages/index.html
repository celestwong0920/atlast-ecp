Evidence Chain Protocol (ECP)

Every AI agent action is recorded as a cryptographically linked chain. SHA-256 hashes + Ed25519 signatures = tamper-proof evidence.

⚡

One-Line Integration

pip install atlast-ecp && atlast run python my_agent.py — zero code changes, instant evidence recording via transparent proxy.

🔒

Privacy by Design

Content never leaves your device. Only hashes are transmitted. Commit-Reveal architecture ensures zero-knowledge proof of work.

⛓️

On-Chain Anchoring

Merkle roots anchored to Base via Ethereum Attestation Service (EAS). Permanent, public, independently verifiable.

📊

Trust Signals

8 behavioral flags (retried, hedged, error, high_latency...) automatically detected. Reliability, efficiency, transparency scores computed.

🌐

Cross-SDK Interop

Python and TypeScript SDKs produce identical hashes. Records from any SDK are verifiable by any other.
